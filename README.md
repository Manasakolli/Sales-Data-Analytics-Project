# Sales Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, MySQL, and Power BI. The objective is to analyze sales data, generate business insights, and visualize key performance metrics through an interactive dashboard.

## Dataset

Sales transaction dataset containing information such as:

* Orders
* Customers
* Products
* Sales
* Profit
* Regions
* Order Dates

## Tools Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* Power BI
* Jupyter Notebook

## Project Steps

### Data Preparation

* Loaded the dataset into Python.
* Cleaned missing values and duplicates.
* Performed data transformation and preprocessing.

### Exploratory Data Analysis (EDA)

* Analyzed sales and profit trends.
* Evaluated product and regional performance.
* Generated visual insights using charts and graphs.

### SQL Analysis

* Imported cleaned data into MySQL.
* Wrote SQL queries to answer business-related questions.
* Performed sales, customer, and profit analysis.

### Dashboard Development

Built an interactive Power BI dashboard featuring:

* Total Sales
* Total Profit
* Sales by Region
* Product Performance
* Monthly Sales Trends
* Key Business KPIs
